# 整数问题 — 子集树求最小差值

## 题目描述

给定一组整数，从里面挑选出一组整数，使得**选中的整数的和**与**剩下的整数的和**的差最小。

**示例：**

> 输入: `arr = [12, 6, 7, 11, 16, 3, 8]`
> 输出: 选中的子集 `{6, 11, 3, 8}`，最小差值为某个具体值
> 解释: 选中一组数，计算 sum(选中) - sum(未选) 的绝对值最小

## 思路分析

这本质上是**子集树问题**：每个元素要么选、要么不选。

**状态设计：**
- `sum` — 当前已选元素的总和
- `r` — 剩余未处理元素的总和（初始为所有元素和，递归时递减）
- 到达叶子节点时，差值 = `|sum - (r_初始 - sum)|` = `|2*sum - total|`

**剪枝策略：**
- 全局变量 `minver` 记录当前最小差值
- 配合限界函数减少无效搜索

## C++ 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int arr[] = {12, 6, 7, 11, 16, 3, 8};
const int length = sizeof(arr) / sizeof(arr[0]);

int x[length] = {0};                // 子集树辅助数组：记录每个元素选(1)或不选(0)
int bestx[length] = {0};            // 记录最优解
unsigned int minver = 0xFFFFFFFF;   // 记录最小差值（初始化为最大值）
int sum = 0;                        // 所选子集的总和
int r = 0;                          // 剩余未处理元素的总和

void func(int i) {
    if (i == length) {              // 到达叶子节点
        int res = abs(sum - (r - sum));  // 选中 - 未选 的差值
        if (res < minver) {
            minver = res;
            for (int j = 0; j < length; ++j)
                bestx[j] = x[j];    // 记录最优子集
        }
        return;
    }
    
    // 选择当前元素（左子树）
    r -= arr[i];
    sum += arr[i];
    x[i] = 1;
    func(i + 1);
    sum -= arr[i];
    r += arr[i];
    
    // 不选当前元素（右子树）
    x[i] = 0;
    func(i + 1);
}

int main() {
    for (int v : arr) r += v;       // 初始化剩余总和
    func(0);
    
    for (int i = 0; i < length; ++i)
        if (bestx[i] == 1)
            cout << arr[i] << " ";
    cout << endl;
    cout << "minver:" << minver << endl;
    return 0;
}
```

## 复杂度分析

| 维度 | 指标 |
|------|------|
| 时间复杂度 | $O(2^n)$，子集树遍历所有叶子节点 |
| 空间复杂度 | $O(n)$，递归栈深度 |

## 易错点 / 拓展思考

- ❗ `r -= arr[i]` 必须在递归左子树之前，不选时再恢复，这个顺序容易错
- ❗ 注意 `minver` 的初始值要足够大（`0xFFFFFFFF`），否则第一次比较无法更新
- ❗ 当前版本没有剪枝，当 n 较大时效率低 — 可以加上限界函数优化
- **拓展 1**：如果要求选中**恰好 n 个**元素（即 2n 选 n），如何修改？
- **拓展 2**：能否用 DP 的方法（子集和问题）求解？
