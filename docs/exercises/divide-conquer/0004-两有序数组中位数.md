# 0004. 寻找两个有序数组的中位数

## 题目描述

给定两个大小分别为 `m` 和 `n` 的升序数组 `nums1` 和 `nums2`。请找出并返回这两个正序数组的**中位数**。

要求算法的时间复杂度为 $O(\log(m+n))$。

**示例 1：**

> 输入: `nums1 = [1,3]`, `nums2 = [2]`
> 输出: `2.0`

**示例 2：**

> 输入: `nums1 = [1,2]`, `nums2 = [3,4]`
> 输出: `2.5`

## 思路分析

核心思想：在**较短数组**上二分搜索，找到一个分割位置 i，使得：

```
nums1[0..i-1] 和 nums2[0..j-1] 组成左半部分
nums1[i..]    和 nums2[j..]    组成右半部分
条件：左半最大值 ≤ 右半最小值，且 i + j = (m+n+1)/2 （左半多一个元素）
```

**中位数公式：**
- 总长度偶数：`(max(左半) + min(右半)) / 2`
- 总长度奇数：`max(左半)`

**为什么二分搜索可行？** 因为 i 增加时 j 会减少（j = k - i），形成了单调关系，可以用二分法找到正确的划分点。

## C++ 代码

```cpp
double findmedian(vector<int> &nums1, int len1, vector<int> &nums2, int len2) {
    // 在较短的数组上二分（保证 len1 <= len2）
    if (len1 > len2)
        return findmedian(nums2, len2, nums1, len1);
    
    int i = 0, j = 0;
    int begin = 0, end = len1;
    int k = (len1 + len2 + 1) / 2;   // 左半部分元素总数
    
    while (begin <= end) {
        i = (begin + end) / 2;        // nums1 左边取 i 个
        j = k - i;                    // nums2 左边取 j 个
        
        if (i > 0 && j < len2 && nums1[i - 1] > nums2[j])
            end = i - 1;              // i 太大，左移
        else if (j > 0 && i < len1 && nums2[j - 1] > nums1[i])
            begin = i + 1;            // i 太小，右移
        else
            break;                    // 找到合适的划分
    }
    
    // 处理边界情况，计算左半最大值
    int left = 0;
    if (i == 0)          left = nums2[j - 1];
    else if (j == 0)     left = nums1[i - 1];
    else                 left = max(nums1[i - 1], nums2[j - 1]);
    
    // 计算右半最小值
    int right = 0;
    if (i == len1)       right = nums2[j];
    else if (j == len2)  right = nums1[i];
    else                 right = min(nums1[i], nums2[j]);
    
    return (len1 + len2) % 2 == 0 ? (left + right) / 2.0 : left;
}
```

## 复杂度分析

| 维度 | 指标 |
|------|------|
| 时间复杂度 | $O(\log(\min(m,n)))$，在较短数组上二分 |
| 空间复杂度 | $O(1)$ |

## 易错点 / 拓展思考

- ❗ 必须**在较短的数组上二分**，否则 j = k - i 可能为负
- ❗ 边界处理最多（i=0, i=len1, j=0, j=len2），四种情况都要覆盖
- ❗ `k = (len1+len2+1)/2` 中的 `+1` 是为了让左半多一个元素，奇数长度时中位数在左半
- **拓展 1**：如果不要求 $O(\log)$ 时间，可以用双指针 $O(m+n)$ 归并解决
- **拓展 2**：这题的思路可以推广到求两个数组的**第 K 大**（Top K 的分治思想）
