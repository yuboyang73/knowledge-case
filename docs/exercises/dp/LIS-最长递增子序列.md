# LIS — 最长递增子序列

## 题目描述

给定一个整数序列，求它的最长**非降**（递增）子序列的长度。

**非降子序列**：序列中的元素可以不连续，但相对顺序不变，且前一个 ≤ 后一个。

**示例：**

> 输入: `[5, 3, 4, 1, 8, 7, 9]`
> 输出: 4
> 解释: 最长非降子序列为 `[3, 4, 7, 9]` 或 `[3, 4, 8, 9]`，长度 4

## 思路分析

### 一维 DP 定义

`dp[i]` 表示以第 `i` 个元素**结尾**的最长非降子序列长度。

**状态转移：**

对于每个 `i`，扫描它前面的所有元素 `j (0..i-1)`：

```
dp[i] = max(1, dp[j] + 1)  条件: arr[j] <= arr[i]
```

初始时每个元素自己构成一个长度为 1 的序列。

```
arr:  [5, 3, 4, 1, 8, 7, 9]
dp:   [1, 1, 2, 1, 3, 3, 4]
         ↓
     3≤4  → dp[2] = dp[1]+1 = 2
```

## C++ 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int arr[] = {5, 3, 4, 1, 8, 7, 9};
    const int length = sizeof(arr) / sizeof(arr[0]);
    
    int dp[length] = {0};
    int maxLen = 0;
    
    for (int i = 0; i < length; i++) {
        dp[i] = 1;                     // 每个数自己是一个长度为 1 的序列
        for (int j = 0; j < i; j++) {
            if (arr[j] <= arr[i] && dp[j] + 1 > dp[i]) {
                dp[i] = dp[j] + 1;     // 接在 arr[j] 后面更优
            }
        }
        maxLen = max(maxLen, dp[i]);   // 更新全局最大值
    }
    
    cout << maxLen << endl;  // 输出 4
    return 0;
}
```

## 复杂度分析

| 维度 | 指标 |
|------|:----:|
| 时间复杂度 | $O(n^2)$，双重循环扫描 |
| 空间复杂度 | $O(n)$，dp 数组 |

## 易错点 / 拓展思考

- ❗ `dp[i]` 的定义是**以 i 结尾**的 LIS 长度，不是前 i 个的 LIS 长度
- ❗ 非降 = 允许相等（`arr[j] <= arr[i]`），严格递增用 `arr[j] < arr[i]`
- ❗ 最终答案是 `max(dp)`，不是 `dp[n-1]`
- **拓展 1**：还有 $O(n \log n)$ 的贪心 + 二分优化解法，用 `tail` 数组维护长度 i 的 LIS 的最小末尾值
- **拓展 2**：如果要求输出具体的 LIS 序列，如何记录路径？
