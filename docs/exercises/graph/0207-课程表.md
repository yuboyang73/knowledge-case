# 0207. 课程表

## 题目描述

你这个学期必须选修 `numCourses` 门课程，记为 `0` 到 `numCourses - 1`。

在选修某些课程之前需要一些先修课程。先修课程按数组 `prerequisites` 给出，其中 `prerequisites[i] = [ai, bi]`，表示如果要学习课程 `ai` 则 **必须** 先学习课程 `bi`。

请你判断是否可能完成所有课程的学习？如果可以，返回 `true`；否则，返回 `false`。

**示例 1：**

> 输入: `numCourses = 2`, `prerequisites = [[1,0]]`
> 输出: `true`
> 解释: 先学 0，再学 1

**示例 2：**

> 输入: `numCourses = 2`, `prerequisites = [[1,0],[0,1]]`
> 输出: `false`
> 解释: 0 和 1 互为先修，形成环

## 思路分析

### 方法一：Kahn 算法（拓扑排序 BFS）

```
1. 建图：邻接表 + 入度数组
2. 将所有入度为 0 的节点入队
3. 出队时计数 cnt++，将其所有邻接节点的入度 -1
   - 若入度变为 0，则入队
4. 如果 cnt == numCourses，说明无环，可以学完
```

### 方法二：DFS 判环

```
状态数组 state[u] = 0（未访问）/ 1（正在访问）/ 2（已访问）
DFS(u):
  state[u]==1 → 发现环，返回 true
  state[u]==2 → 已处理过无环，返回 false
  state[u]=1 → 递归邻居 → state[u]=2
  若任何邻居有环则返回 true
```

## C++ 代码

```cpp
// 方法一：Kahn 算法（拓扑排序）
class Solution {
public:
    bool canFinish(int numCourses, vector<vector<int>>& prerequisites) {
        vector<vector<int>> adj(numCourses);  // 邻接表
        vector<int> din(numCourses, 0);       // 入度
        
        for (auto &pre : prerequisites) {
            int a = pre[0], b = pre[1];
            adj[b].push_back(a);   // b → a
            din[a]++;               // a 的入度 +1
        }
        
        queue<int> q;
        for (int i = 0; i < numCourses; i++)
            if (!din[i]) q.push(i);
        
        int cnt = 0;
        while (!q.empty()) {
            int x = q.front(); q.pop();
            cnt++;
            for (int y : adj[x])
                if (--din[y] == 0) q.push(y);
        }
        return cnt == numCourses;  // 全部学完 = 无环
    }
};

// 方法二：DFS 判环
class Solution2 {
public:
    bool canFinish(int numCourses, vector<vector<int>>& prerequisites) {
        vector<vector<int>> adj(numCourses);
        vector<int> state(numCourses, 0);  // 0=未访问, 1=访问中, 2=已访问
        
        for (auto &pre : prerequisites)
            adj[pre[1]].push_back(pre[0]);
        
        for (int i = 0; i < numCourses; i++)
            if (dfs(i, adj, state)) return false;
        return true;
    }
    
    bool dfs(int u, vector<vector<int>>& adj, vector<int>& state) {
        if (state[u] == 1) return true;   // 发现环
        if (state[u] == 2) return false;  // 已处理过，无环
        
        state[u] = 1;  // 正在访问
        for (int v : adj[u])
            if (dfs(v, adj, state)) return true;
        state[u] = 2;  // 访问完成
        
        return false;
    }
};
```

## 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 |
|------|:---------:|:---------:|
| Kahn 算法 | $O(V + E)$ | $O(V + E)$ |
| DFS 判环 | $O(V + E)$ | $O(V + E)$ |

## 易错点 / 拓展思考

- ❗ Kahn 算法中，建图方向是 `b → a`（先修 → 后修），入度 `din[a]++`
- ❗ `t.size()` 返回 `size_t`，不能与 -1 比较，要先用 int 变量缓存
- ❗ DFS 的三种状态缺一不可，只有 0/1（未访问/已访问）无法区分当前路径上的环
- **拓展 1**：如果要求返回**具体的课程顺序**（LeetCode 210），Kahn 算法中记录出队顺序即可
- **拓展 2**：如果图不是 DAG，是否有拓扑排序的替代方案？
