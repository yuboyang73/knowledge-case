# 0994. 腐烂的橘子

## 题目描述

在给定的 `m x n` 网格 `grid` 中，每个单元格可以有以下三个值之一：

- 值 `0` — 空单元格
- 值 `1` — 新鲜橘子
- 值 `2` — 腐烂的橘子

每分钟，腐烂的橘子 **周围 4 个方向上相邻** 的新鲜橘子都会腐烂。

返回直到单元格中没有新鲜橘子为止所必须经过的最小分钟数。如果不可能全部腐烂，返回 `-1`。

**示例 1：**

```
输入: grid = [[2,1,1],[1,1,0],[0,1,1]]
输出: 4
```

**示例 2：**

```
输入: grid = [[0,2]]
输出: 0（一开始就没有新鲜橘子）
```

**示例 3：**

```
输入: grid = [[2,1,1],[0,1,1],[1,0,1]]
输出: -1（左下角的橘子永远无法被感染）
```

## 思路分析

**多源 BFS** 的典型应用：

```
1. 遍历网格，将所有腐烂橘子入队，统计新鲜橘子数量 fresh
2. BFS 分层遍历：
   - 每层代表一分钟
   - 对每个腐烂橘子，四方向扩展
   - 若遇到新鲜橘子，将其腐烂并入队，fresh--
3. 循环结束：fresh==0 → 返回分钟数，否则返回 -1
```

**方向数组写法（三种风格等价）：**

```cpp
// 风格 1：两个数组
int dx[4] = {1, -1, 0, 0};
int dy[4] = {0, 0, 1, -1};

// 风格 2：pair 向量
vector<pair<int,int>> dir = {{-1,0},{1,0},{0,-1},{0,1}};

// 风格 3：一维数组（推荐）
int d[5] = {-1, 0, 1, 0, -1};
for (int k = 0; k < 4; k++) {
    int nx = x + d[k];
    int ny = y + d[k + 1];
}
```

## C++ 代码

```cpp
class Solution {
public:
    int orangesRotting(vector<vector<int>>& grid) {
        int n = grid.size(), m = grid[0].size();
        queue<pair<int, int>> q;
        int fresh = 0;
        int d[5] = {-1, 0, 1, 0, -1};
        
        // 初始化：腐烂橘子入队，统计新鲜橘子
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < m; j++) {
                if (grid[i][j] == 2)
                    q.push({i, j});
                else if (grid[i][j] == 1)
                    fresh++;
            }
        }
        
        if (fresh == 0) return 0;  // 没有新鲜橘子
        
        int minutes = 0;
        while (!q.empty() && fresh > 0) {
            int size = q.size();
            minutes++;              // 每层 = 1 分钟
            
            for (int i = 0; i < size; i++) {
                auto [x, y] = q.front(); q.pop();
                
                for (int k = 0; k < 4; k++) {
                    int nx = x + d[k];
                    int ny = y + d[k + 1];
                    if (nx >= 0 && nx < n && ny >= 0 && ny < m && grid[nx][ny] == 1) {
                        grid[nx][ny] = 2;   // 腐烂
                        q.push({nx, ny});
                        fresh--;
                    }
                }
            }
        }
        
        return fresh == 0 ? minutes : -1;
    }
};
```

## 复杂度分析

| 维度 | 指标 |
|------|:----:|
| 时间复杂度 | $O(n \times m)$，每个格子最多入队一次 |
| 空间复杂度 | $O(n \times m)$，队列存储 |

## 易错点 / 拓展思考

- ❗ 注意**分层**：`int size = q.size()` 要在每次循环开始记录，不能放在 for 条件里（q.size() 会变）
- ❗ BFS 结束时 `fresh > 0` 说明有橘子永远无法被感染，返回 -1
- ❗ 方向数组 `d[5]` 中 `d[k]` 和 `d[k+1]` 配对，比两个数组更简洁
- **拓展 1**：如果橘子腐烂后可以**向八个方向**（包括斜角）扩散，如何修改？
- **拓展 2**：如果每个橘子腐烂需要的时间不同（加权 BFS），如何求解？
