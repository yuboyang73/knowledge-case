# 硬币问题 — 贪心

## 题目描述

现有面值为 1、3、5 分的硬币，每种数量无限。给定一个目标价值 c，问组成该价值所需的最少硬币数量。

**示例：**

> 输入: 硬币面值 `[1, 3, 5]`，目标 `c = 11`
> 输出: 3（5 + 5 + 1）或（5 + 3 + 3）

## 思路分析

**贪心策略**：每次都选**当前能放下的最大面值**硬币。

```
c = 11, 面值 [5, 3, 1]（降序）
① 选 5 → c = 6
② 选 5 → c = 1
③ 选 1 → c = 0
完成，共 3 枚硬币
```

> 本题面值特殊（1 分保证有解，且大面值都是小面值的倍数），贪心能得到最优解。
> 如果面值是 [1, 3, 4]，目标 c = 6，贪心会得到 4+1+1=3 枚，但最优是 3+3=2 枚，此时贪心失效，需要用 DP。

## C++ 代码

```cpp
int main() {
    int arr[] = {1, 3, 5};
    int size = sizeof(arr) / sizeof(arr[0]);
    int c = 11;
    
    // 按面值降序排列
    sort(arr, arr + size, [](int a, int b) { return a > b; });
    
    int idx = 0;    // 当前面值下标
    int cnt = 0;    // 硬币计数
    while (c > 0) {
        if (c >= arr[idx]) {
            c -= arr[idx];     // 能放就放
            cnt++;
        } else {
            idx++;             // 放不下就换小面值
        }
    }
    cout << cnt << endl;       // 输出：3
    return 0;
}
```

## 复杂度分析

| 维度 | 指标 |
|------|------|
| 时间复杂度 | $O(n)$，n 为不同面值数 |
| 空间复杂度 | $O(1)$ |

## 易错点 / 拓展思考

- ❗ 贪心**不一定保证全局最优**！硬币面值必须满足「大面值是小面值的倍数」时贪心才正确
- ❗ 降序排序后，idx 递增而不是递减（从大往小试）
- **拓展 1**：如果面值是 [1, 3, 4]，目标 6，贪心得到 3 枚，但最优是 2 枚（3+3）。此时应改用 DP
- **拓展 2**：完全背包的 DP 解法：`dp[i] = min(dp[i], dp[i - coin] + 1)`
