# 0049. 字母异位词分组

## 题目描述

给你一个字符串数组，请你将 **字母异位词** 组合在一起。可以按任意顺序返回结果列表。

**字母异位词** 指由相同字母重排列形成的字符串（包括相同的字符串）。

**示例 1：**

> 输入: `strs = ["eat", "tea", "tan", "ate", "nat", "bat"]`
> 输出: `[["bat"],["nat","tan"],["ate","eat","tea"]]`

**示例 2：**

> 输入: `strs = [""]`
> 输出: `[[""]]`

**示例 3：**

> 输入: `strs = ["a"]`
> 输出: `[["a"]]`

## 思路分析

字母异位词排序后会得到**相同的字符串**，利用这一性质构建哈希映射：

```
"eat" → sort → "aet" ─┐
"tea" → sort → "aet" ─┼──→ { "aet": ["eat", "tea", "ate"] }
"ate" → sort → "aet" ─┘
"tan" → sort → "ant" ──→ { "ant": ["tan", "nat"] }
```

- key = 字符串排序后的结果
- value = 原始字符串数组
- 遍历 strs，将每个字符串归入对应的分组

## C++ 代码

```cpp
#include <bits/stdc++.h>
using namespace std;

vector<vector<string>> groupAnagrams(vector<string> &strs) {
    vector<vector<string>> res;
    unordered_map<string, vector<string>> map;
    
    for (auto &s : strs) {
        string key = s;
        sort(key.begin(), key.end());  // 排序后作为 key
        map[key].push_back(s);         // 将原字符串加入分组
    }
    
    for (auto &m : map) {
        res.push_back(m.second);       // 收集所有分组
    }
    return res;
}
```

## 复杂度分析

| 维度 | 指标 |
|------|------|
| 时间复杂度 | $O(n \cdot k \log k)$，$n$ 为字符串个数，$k$ 为最长字符串长度 |
| 空间复杂度 | $O(n \cdot k)$，哈希表存储所有字符串 |

## 易错点 / 拓展思考

- ❗ key 不一定要用排序，也可以用**字符计数数组**转为字符串，时间可优化到 $O(n \cdot k)$
- ❗ `map[key]` 的 value 是 `vector<string>`，要用 `push_back` 插入，不能直接赋值
- 思考：如果用 `unordered_map<string, vector<string>>`，自定义类型做 key 需要注意什么？
