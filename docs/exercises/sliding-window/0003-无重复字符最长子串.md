# 0003. 无重复字符的最长子串

## 题目描述

给定一个字符串 `s`，请你找出其中不含有重复字符的 **最长子串** 的长度。

**示例：**

> 输入: `s = "abcabcbb"`
> 输出: 3
> 解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

## 思路分析

### 方法一：哈希 + 滑动窗口

用 `unordered_set` 维护滑动窗口内的字符：

```
s = "a b c a b c b b"

[a]           → 窗口: {a}, len=1
[a,b]         → 窗口: {a,b}, len=2
[a,b,c]       → 窗口: {a,b,c}, len=3
[a,b,c,a]     → 发现 a 重复，移除左边界 a → [b,c,a], len=3
[b,c,a,b]     → 发现 b 重复，移除左边界 b → [c,a,b], len=3
... 以此类推
```

### 方法二：跳跃滑动窗口（优化）

用数组记录每个字符**下一次出现的合法位置**，左边界直接跳到该位置，不需要逐步移动：

```cpp
left = max(left, index[s[right]]);   // 直接跳跃
index[s[right]] = right + 1;          // 记录该字符下一次的合法位置
```

## C++ 代码

```cpp
// 方法一：哈希 + 双指针（易懂）
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_set<char> se;
        int n = s.size();
        int left = 0, maxLen = 0;
        
        for (int right = 0; right < n; ++right) {
            // 重复了，收缩左边界
            while (se.find(s[right]) != se.end()) {
                se.erase(s[left++]);
            }
            se.insert(s[right]);
            maxLen = max(maxLen, right - left + 1);
        }
        return maxLen;
    }
};
```

```cpp
// 方法二：跳跃滑动窗口（优化）
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        int index[128] = {0};     // ASCII 字符集
        int left = 0, maxLen = 0;
        
        for (int right = 0; right < s.size(); ++right) {
            char ch = s[right];
            left = max(left, index[ch]);    // 左边界直接跳到合法位置
            maxLen = max(maxLen, right - left + 1);
            index[ch] = right + 1;          // 记录该字符的下一个合法位置
        }
        return maxLen;
    }
};
```

## 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 |
|------|:---------:|:---------:|
| 哈希 + 双指针 | $O(n)$ | $O(\Sigma)$，$\Sigma$ 为字符集大小 |
| 跳跃滑动窗口 | $O(n)$ | $O(\Sigma)$ |

## 易错点 / 拓展思考

- ❗ 方法一中 `se.erase(s[left++])` 是不断移除左边界字符直到窗口内不再包含重复字符，不是只移除一次
- ❗ 方法二中 `index[ch]` 初始值 0，但 left 也从 0 开始，所以 `left=max(left,index[ch])` 天然处理了未出现的情况
- ❗ ASCII 128 数组只适用于英文字母/数字/符号，Unicode 字符需要用 unordered_map
- **拓展 1**：如果允许**最多 K 个重复字符**（至多包含 K 个不同字符），滑动窗口如何调整？
- **拓展 2**：找到**所有**无重复字符的子串位置，而非仅最大长度
