# 0098. 验证二叉搜索树

## 题目描述

给你一个二叉树的根节点 `root`，判断其是否是一个有效的二叉搜索树（BST）。

**有效 BST 定义：**
- 节点的左子树只包含**小于**当前节点的数
- 节点的右子树只包含**大于**当前节点的数
- 所有左子树和右子树自身也必须是二叉搜索树

**示例：**

```
输入: root = [2,1,3]
输出: true
```

```
输入: root = [5,1,4,null,null,3,6]
输出: false
说明: 根节点值为 5，右子节点值为 4，不满足 BST 定义
```

## 思路分析

### 方法一：中序遍历递归

BST 的**中序遍历是严格升序的**。利用全局变量 `pre` 记录前一个访问的节点：

```
中序遍历（LVR）：
  先遍历左子树
  检查当前值是否 > pre 的值（pre 是左子树的最右节点）
  更新 pre = 当前节点
  再遍历右子树
```

### 方法二：区间递归

```
isValidBST(root, min, max):
  root->val 必须在 (min, max) 范围内
  左子树的范围: (min, root->val)
  右子树的范围: (root->val, max)
```

## C++ 代码

```cpp
// 方法一：中序遍历递归（利用 BST 升序性质）
class Solution {
public:
    bool isBST(TreeNode* cur, TreeNode*& pre) {
        if (!cur) return true;
        
        if (!isBST(cur->left, pre)) return false;  // L：左子树不是 BST
        
        if (pre && cur->val <= pre->val)            // V：检查升序
            return false;
        pre = cur;                                  // 更新 pre
        
        return isBST(cur->right, pre);              // R：右子树
    }
    
    bool isValidBST(TreeNode* root) {
        TreeNode* pre = nullptr;
        return isBST(root, pre);
    }
};
```

```cpp
// 方法二：区间递归
class Solution {
public:
    bool isValidBST(TreeNode* root, long min = LONG_MIN, long max = LONG_MAX) {
        if (!root) return true;
        if (root->val <= min || root->val >= max)
            return false;
        return isValidBST(root->left, min, root->val)
            && isValidBST(root->right, root->val, max);
    }
};
```

## 复杂度分析

| 维度 | 指标 |
|------|:----:|
| 时间复杂度 | $O(n)$，每个节点访问一次 |
| 空间复杂度 | $O(h)$，递归栈深度 |

## 易错点 / 拓展思考

- ❗ 不能只检查 `root->left->val < root->val` 和 `root->right->val > root->val`，还要保证左子树的所有节点 < 根，右子树的所有节点 > 根
- ❗ 中序遍历法中的 `pre` 需要用引用传递（`TreeNode*&`），否则递归中 pre 修改不会传递到上层
- ❗ 边界值：题目可能有 `INT_MIN` / `INT_MAX` 的值，所以区间递归用 `long` 或 `LONG_MIN/LONG_MAX`
- **拓展 1**：如何利用**迭代**中序遍历（显式栈）验证 BST？
- **拓展 2**：如何找到 BST 中第 K 大的元素？（中序遍历的变体）
