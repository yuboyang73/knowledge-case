# 0101. 对称二叉树

## 题目描述

给你一个二叉树的根节点 `root`，检查它是否轴对称。

**示例：**

```
输入: root = [1,2,2,3,4,4,3]
输出: true

     1
   /   \
  2     2
 / \   / \
3   4 4   3
```

```
输入: root = [1,2,2,null,3,null,3]
输出: false

     1
    / \
   2   2
    \   \
     3    3
```

## 思路分析

**对称的条件**：左子树的左孩子和右子树的右孩子相等，且左子树的右孩子和右子树的左孩子相等。

```
func(left, right):
  left==null && right==null → true
  left==null || right==null → false
  left->val != right->val  → false
  return func(left->left, right->right) && func(left->right, right->left)
```

## C++ 代码

```cpp
class Solution {
public:
    bool func(TreeNode* left, TreeNode* right) {
        if (!left && !right) return true;     // 都为空 → 对称
        if (!left || !right) return false;    // 一个为空 → 不对称
        
        // 当前节点值相等 && 外侧对称 && 内侧对称
        return left->val == right->val
            && func(left->left, right->right)   // 左左 vs 右右
            && func(left->right, right->left);  // 左右 vs 右左
    }
    
    bool isSymmetric(TreeNode* root) {
        if (!root) return true;
        return func(root->left, root->right);
    }
};
```

## 复杂度分析

| 维度 | 指标 |
|------|:----:|
| 时间复杂度 | $O(n)$，每个节点访问一次 |
| 空间复杂度 | $O(h)$，递归栈深度 |

## 易错点 / 拓展思考

- ❗ 比较的是 `left->left` 与 `right->right`（外侧），以及 `left->right` 与 `right->left`（内侧），注意不要配错对
- ❗ 空节点检查必须用 `!left && !right`（都空）和 `!left || !right`（一个空）分开处理，不能合并
- **拓展 1**：如何用**迭代**（栈/队列）实现？
- **拓展 2**：如何判断一棵树是否是**中心对称**（绕根节点旋转 180° 后与自身重合）？
