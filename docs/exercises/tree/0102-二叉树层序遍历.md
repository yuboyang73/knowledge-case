# 0102. 二叉树的层序遍历

## 题目描述

给你二叉树的根节点 `root`，返回其节点值的 **层序遍历**（即逐层地，从左到右访问所有节点，每层一个数组）。

**示例：**

```
输入: root = [3,9,20,null,null,15,7]
输出: [[3],[9,20],[15,7]]

    3
   / \
  9  20
    /  \
   15   7
```

## 思路分析

### 方法一：队列 BFS

BFS 经典应用，用 `size` 记录当前层的节点数：

```
1. root 入队
2. 循环直到队列为空：
    size = q.size()  （当前层的节点数）
    循环 size 次：
      出队 → 放入当前层数组
      左孩子入队、右孩子入队
    当前层数组加入结果
```

### 方法二：递归 DFS

用 `k` 记录当前层数，递归时传入 `k+1`：

```
如果 res.size() <= k，说明第一次到达第 k 层，新建数组
将 root->val 放入 res[k]
递归左子树(k+1) 和 右子树(k+1)
```

## C++ 代码

```cpp
// 方法一：队列 BFS
class Solution {
public:
    vector<vector<int>> levelOrder(TreeNode* root) {
        if (!root) return {};
        
        vector<vector<int>> res;
        queue<TreeNode*> q;
        q.push(root);
        
        while (!q.empty()) {
            int size = q.size();           // 当前层的节点数
            vector<int> cur;
            
            for (int i = 0; i < size; i++) {
                TreeNode* x = q.front(); q.pop();
                cur.push_back(x->val);
                if (x->left)  q.push(x->left);
                if (x->right) q.push(x->right);
            }
            res.push_back(cur);
        }
        return res;
    }
};
```

```cpp
// 方法二：递归 DFS
class Solution {
public:
    void yj(TreeNode* root, vector<vector<int>>& res, int k) {
        if (!root) return;
        if (res.size() <= k)
            res.push_back({});           // 第一次到达第 k 层
        res[k].push_back(root->val);
        yj(root->left, res, k + 1);
        yj(root->right, res, k + 1);
    }
    
    vector<vector<int>> levelOrder(TreeNode* root) {
        vector<vector<int>> res;
        yj(root, res, 0);
        return res;
    }
};
```

## 复杂度分析

| 方法 | 时间复杂度 | 空间复杂度 |
|------|:---------:|:---------:|
| BFS 队列 | $O(n)$ | $O(n)$ |
| DFS 递归 | $O(n)$ | $O(n)$ 递归栈 |

## 易错点 / 拓展思考

- ❗ BFS 中 `size` 必须在 for 循环前固定，不能写成 `i < q.size()`（q.size() 会动态变化）
- ❗ 递归方法中 `res.size() <= k` 说明是第一次访问该层，需要新建数组
