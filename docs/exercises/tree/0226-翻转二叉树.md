# 0226. 翻转二叉树

## 题目描述

给你一棵二叉树的根节点 `root`，翻转这棵二叉树，并返回其根节点。

翻转即**交换每个节点的左右子树**。

**示例：**

```
输入: root = [4,2,7,1,3,6,9]
输出: [4,7,2,9,6,3,1]

      4                 4
     / \               / \
    2   7      →      7   2
   / \ / \           / \ / \
  1  3 6 9          9  6 3  1
```

## 思路分析

**递归思路**：对于每个节点，交换其左右孩子，然后递归地对左右子树执行同样的操作。

```
invertTree(root):
  if root == null: return
  swap(root->left, root->right)
  invertTree(root->left)
  invertTree(root->right)
  return root
```

可以用前序、后序、层序遍历实现，但**中序遍历**不行（会重复翻转左边的子树）。

## C++ 代码

```cpp
class Solution {
public:
    TreeNode* invertTree(TreeNode* root) {
        if (!root) return nullptr;
        
        swap(root->left, root->right);       // 交换当前节点的左右孩子
        
        invertTree(root->left);              // 递归翻转左子树
        invertTree(root->right);             // 递归翻转右子树
        
        return root;
    }
};
```

## 复杂度分析

| 维度 | 指标 |
|------|:----:|
| 时间复杂度 | $O(n)$，每个节点访问一次 |
| 空间复杂度 | $O(h)$，h 为树高，最坏 $O(n)$ |

## 易错点 / 拓展思考

- ❗ 如果使用**中序遍历**（左→根→右），右子树翻转后实际翻转的是左子树的左孩子，会出现错误
- ❗ 交换的是**节点**，不是节点的值（虽然效果相同，但 swap 节点更符合语义）
- **拓展 1**：如何用**非递归**（迭代 + 栈）实现？
- **拓展 2**：如何判断一棵树翻转后和另一棵树相同？（与 0101 对称二叉树结合思考）
