# 0543. 二叉树的直径

## 题目描述

给你一棵二叉树的根节点，返回该树的 **直径**。

二叉树的 **直径** 是指树中任意两个节点之间最长路径的**长度**。这条路径可能经过也可能不经过根节点。两节点之间的路径长度由它们之间的边数表示。

**示例：**

```
输入: root = [1,2,3,4,5]
输出: 3

      1       路径 [4,2,1,3] 或 [5,2,1,3]
     / \      长度 = 3（3 条边）
    2   3
   / \
  4   5
```

## 思路分析

**核心思想**：对于每个节点，它的直径 = 左子树深度 + 右子树深度。

**DFS 后序遍历：**
```
1. 递归计算左子树深度 left
2. 递归计算右子树深度 right
3. 更新全局最大直径 ans = max(ans, left + right)
4. 返回当前节点的深度 = max(left, right) + 1
```

```
        1
       / \       左子树深度=2 (2→4/5)，右子树深度=1 (3)
      2   3      当前节点直径 = 2 + 1 = 3
     / \
    4   5        叶子节点深度 = 1
```

## C++ 代码

```cpp
class Solution {
public:
    int dfs(TreeNode* root, int& ans) {
        if (!root) return 0;
        
        int left = dfs(root->left, ans);    // 左子树深度
        int right = dfs(root->right, ans);  // 右子树深度
        
        ans = max(ans, left + right);       // 更新直径
        
        return max(left, right) + 1;        // 返回当前节点的深度
    }
    
    int diameterOfBinaryTree(TreeNode* root) {
        int ans = 0;
        dfs(root, ans);
        return ans;
    }
};
```

## 复杂度分析

| 维度 | 指标 |
|------|:----:|
| 时间复杂度 | $O(n)$，每个节点访问一次 |
| 空间复杂度 | $O(h)$，递归栈深度 |

## 易错点 / 拓展思考

- ❗ 直径 = **边数**，不是节点数。如示例中路径有 3 条边，答案为 3
- ❗ `ans` 要用引用传递，或者定义为类的成员变量，否则修改不会回溯
- ❗ 直径不一定经过根节点，所以需要在遍历过程中持续更新最大值
- **拓展 1**：如果要求输出具体的路径（节点序列），如何记录？
- **拓展 2**：N 叉树的直径如何计算？原理类似，只是子树从 2 个变为多个
