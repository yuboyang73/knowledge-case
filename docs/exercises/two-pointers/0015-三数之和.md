# 0015. 三数之和

## 题目描述

给你一个整数数组 `nums`，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k`，同时还满足 `nums[i] + nums[j] + nums[k] == 0`。返回所有和为 0 且不重复的三元组。

**示例：**

> 输入: `nums = [-1,0,1,2,-1,-4]`
> 输出: `[[-1,-1,2],[-1,0,1]]`

**注意：** 答案中不可以包含重复的三元组。

## 思路分析

**排序 + 左右双指针**是标准解法：

```
1. 排序（O(nlogn)）
2. 固定第一个数 i，用左右双指针在 i+1..n-1 范围内找两数之和 = -nums[i]
3. 去重：跳过重复元素
```

```
排序后: [-4, -1, -1, 0, 1, 2]
固定 -4 → 在 [-1,-1,0,1,2] 中找两数和 = 4 → 不存在
固定 -1 → 在 [-1,0,1,2] 中找两数和 = 1 → 找到 (-1,2),(0,1)
         → 得到 [-1,-1,2] 和 [-1,0,1]
固定 0  → 在 [1,2] 中找两数和 = 0 → 不存在
```

## C++ 代码

```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> res;
        sort(nums.begin(), nums.end());
        int n = nums.size();
        
        for (int i = 0; i < n - 2; i++) {
            if (i > 0 && nums[i] == nums[i - 1])
                continue;           // 跳过重复的 i
            
            int l = i + 1, r = n - 1;
            int target = -nums[i];
            
            while (l < r) {
                int sum = nums[l] + nums[r];
                if (sum == target) {
                    res.push_back({nums[i], nums[l], nums[r]});
                    // 跳过重复元素
                    while (l < r && nums[l] == nums[l + 1]) l++;
                    while (l < r && nums[r] == nums[r - 1]) r--;
                    l++, r--;
                } else if (sum > target) {
                    r--;
                } else {
                    l++;
                }
            }
        }
        return res;
    }
};
```

## 复杂度分析

| 维度 | 指标 |
|------|:----:|
| 时间复杂度 | $O(n^2)$，外层循环 $O(n)$，内层双指针 $O(n)$ |
| 空间复杂度 | $O(\log n)$ 到 $O(n)$，取决于排序实现 |

## 易错点 / 拓展思考

- ❗ 去重有三处：跳过重复的 `i`、跳过重复的 `l`、跳过重复的 `r`
- ❗ 外层循环 `i < n-2`，因为至少需要三个元素
- ❗ `target = -nums[i]` 而不是 `0 - nums[i]`（两者等价但前更直观）
