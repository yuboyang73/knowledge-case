## 🧭 如何使用

::: tip 理论先行
每个理论模块独立成文，包含概念解析、图解推导和核心代码模板。  
末尾 **配套练习** 表格直接链接到对应习题。
:::

::: tip 习题实战
每道习题独立成文，严格遵循 **题目 → 思路 → 代码 → 复杂度 → 易错点** 的五段式结构。  
头部 `theory_id` 标注前置理论，方便回查。
:::

::: tip 双向跳转
理论 ⇄ 习题 **双向关联**，无论从哪一侧出发都能找到另一侧。
:::
