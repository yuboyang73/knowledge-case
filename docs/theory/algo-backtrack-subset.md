# 回溯算法 — 子集树

## 概念模型

子集树是用于求解**子集类问题**的树形模型，本质是一棵**完全二叉树**：

- 每一层对应集合中的一个元素，左分支 = 选，右分支 = 不选
- 叶子节点对应所有元素的选/不选组合，即原集合的一个子集
- 节点总数 $2^{n+1} - 1$，叶子数 $2^n$

```
                   根
                  /   \
               选1    不选1
              /   \    /   \
           选2  不选2 选2  不选2
           ...  ...  ...  ...
```

## 核心框架

```cpp
void func(int arr[], int i, int length) {
    if (i == length) {          // 到达叶子节点
        // 处理当前选中的子集
        for (int j = 0; j < length; j++)
            if (x[j]) cout << arr[j] << " ";
        cout << endl;
        return;
    }
    
    x[i] = 1;                   // 选择当前元素
    func(arr, i + 1, length);   // 左子树递归
    
    x[i] = 0;                   // 不选当前元素
    func(arr, i + 1, length);   // 右子树递归
}
```

## 题目关键词

- 从一堆东西里 **选** 任意个 / **选** 若干个 / 求满足条件的组合
- 每个元素**只能用一次**
- **不考虑顺序**（选 {1,2} 和 {2,1} 算同一个）

## 剪枝优化

在递归过程中使用**限界函数**剪掉不可能产生最优解的分支：

```cpp
// 经典剪枝模式
if (当前已选价值 + 剩余物品价值 > 当前最优解) {
    func(i + 1);  // 才有必要继续搜索
}
```

## 时间复杂度

- 无剪枝：$O(2^n)$，每个元素选或不选
- 剪枝后：理论不变，但实际运行大幅减少
