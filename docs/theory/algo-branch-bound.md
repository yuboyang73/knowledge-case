# 分支限界算法

## 与回溯算法的区别

| 维度 | 回溯算法 | 分支限界法 |
|------|---------|-----------|
| 求解目标 | 所有解 / 满足条件的解 | **一个解** / **最优解** |
| 搜索方式 | **深度优先** DFS | **广度优先** BFS / 最小耗费优先 |
| 节点扩展 | 产生一个孩子就继续递归 | 一次性产生**所有孩子** |
| 辅助结构 | 递归栈 | 队列 / 优先级队列 |

## 基本概念

```mermaid
flowchart LR
    A[扩展节点] -->|产生所有儿子| B[活结点表]
    B -->|筛选| C[下一個扩展节点]
    C -->|产生儿子| A
```

**关键术语：**

| 术语 | 定义 |
|------|------|
| **扩展节点** | 正在产生儿子的节点 |
| **活结点** | 自身已生成但儿子尚未全部生成的节点 |
| **死结点** | 所有儿子已产生的节点 |
| **限界函数** | 在活结点处计算函数值，选择最有利的节点扩展 |

## 两种实现方式

### 1. FIFO 队列式

```
1. 根结点入队（唯一活结点）
2. 取队首为扩展节点
3. 从左到右产生所有孩子
4. 满足约束的孩子入队
5. 重复 2→4 直到找到解或队列为空
```

### 2. 优先级队列式

```
1. 根结点入优先级队列
2. 取优先级最高（最有利）的节点为扩展节点
3. 从左到右产生所有孩子
4. 满足约束的孩子计算优先级并入队
5. 重复 2→4 直到找到解或队列为空
```

## 适用场景

- 旅行商问题（TSP）
- 装载问题（集装箱装船）
- 0-1 背包问题（求最优解而非所有解）
- 单源最短路径

| 问题 | 推荐算法 |
|------|---------|
