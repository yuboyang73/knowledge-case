# DFS 与 BFS 模板

## 一、DFS（深度优先搜索）

### 通用模板

```cpp
void dfs(/* 参数用来表示当前状态 */) {
    if (/* 到达终点状态 */) {
        // 根据题意处理结果
        return;
    }
    if (/* 越界或不合法状态 */)
        return;
    if (/* 特殊条件 - 剪枝 */)
        return;
    
    for (/* 扩展方式 */) {
        if (/* 扩展方式所达到状态合法 */) {
            修改操作;   // 根据题意标记或累加
            标记;
            dfs();
            还原标记;   // 回溯
        }
    }
}
```

### 经典 DFS 应用

| 问题 | DFS 角色 |
|------|---------|
| 岛屿数量（Flood Fill） | 四方向扩展标记联通分量 |
| 全排列 | 回溯搜索所有排列 |
| N皇后 | 排列树 + 约束判断 |
| 子集和问题 | 子集树 + 剪枝 |

## 二、BFS（广度优先搜索）

### 通用模板

```cpp
#include <queue>

queue<状态类型> q;
q.push(初始状态);

while (!q.empty()) {
    auto cur = q.front();
    q.pop();
    
    for (/* 枚举 cur 的所有可达状态 next */) {
        if (/* next 合法 */) {
            标记 visited;
            q.push(next);
        }
    }
}
```

### BFS 特点

- **最短路径**：BFS 首次到达目标时的路径长度一定最短
- **层序遍历**：可用 `int size = q.size()` 分层处理
- 需要 `visited` 数组防止重复访问

### 经典 BFS 应用

| 问题 | BFS 角色 |
|------|---------|
| 迷宫最短路径 | 四方向 BFS 找最短 |
| 奇怪电梯 | BFS 在抽象状态空间搜索 |
| 腐烂的橘子 | 多源 BFS + 分层计时 |
| 二叉树层序遍历 | 队列分层遍历 |

## 三、DFS vs BFS 对比

| 维度 | DFS | BFS |
|------|-----|-----|
| 数据结构 | 栈（递归栈） | 队列 |
| 空间复杂度 | O(深度) | O(宽度) |
| 最短路径 | ❌ 不一定 | ✅ 首次到达即最短 |
| 实现方式 | 递归 / 显式栈 | 显式队列 |
| 适用场景 | 所有解、排列组合 | 最短路径、层序、连通性 |
