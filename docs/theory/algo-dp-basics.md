# 动态规划基础

## DP 三要素

| 要素 | 说明 | 示例（LCS） |
|------|------|------------|
| **状态定义** | `dp[i][j]` 表示什么含义 | `dp[i][j]` = s1前i个与s2前j个的LCS长度 |
| **状态转移** | 如何从子问题推导当前问题 | 见下方公式 |
| **初始条件** | 递归的 base case / dp 数组初始值 | `dp[0][j] = dp[i][0] = 0` |

## 通用解题步骤

```
1. 确定状态定义（一维/二维/多维）
2. 推导状态转移方程
3. 确定初始条件和边界
4. 确定计算顺序（自顶向下递归 / 自底向上递推）
5. 优化（滚动数组、空间压缩）
```

## 递归 → 递推优化

**自顶向下（记忆化搜索）：**
```cpp
vector<int> memo(n, -1);
int dp(int i) {
    if (i == 0) return base;
    if (memo[i] != -1) return memo[i];
    return memo[i] = dp(i-1) + ...;
}
```

**自底向上（递推）：**
```cpp
vector<int> dp(n);
dp[0] = base;
for (int i = 1; i < n; i++)
    dp[i] = dp[i-1] + ...;
```

## DP vs 分治 vs 贪心

| 算法 | 子问题独立性 | 子问题重叠 | 典型时间复杂度 |
|------|:-----------:|:---------:|:------------:|
| 分治 | ✅ 互相独立 | ❌ 无重叠 | O(nlogn) |
| 动态规划 | ❌ 不独立 | ✅ 大量重叠 | O(n²)/O(n) |
| 贪心 | — | — | O(n) / O(nlogn) |

**判断是否为 DP 问题的关键：** 最优子结构 + 重叠子问题。
