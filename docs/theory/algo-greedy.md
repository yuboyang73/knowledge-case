# 贪心算法

## 基本思想

贪心算法**总是做出当前看来最好的选择**，并不从整体最优考虑。

> 虽然贪心不能对所有问题得到整体最优解，但对很多问题它能产生整体最优解或趋近最优解。

## 适用条件

- **最优子结构**：问题的最优解包含子问题的最优解
- **贪心选择性质**：每一步的局部最优选择能导致全局最优解

## 题目关键

**最大、最小、最多、最少、至多、至少** — 出现这些字眼时优先考虑贪心。

## 贪心 vs 动态规划

| 维度 | 贪心 | 动态规划 |
|------|------|---------|
| 决策方式 | 自顶向下，一步一选 | 自底向上，状态转移 |
| 子问题重叠 | 不要求 | 要求 |
| 证明方式 | 贪心选择 + 最优子结构 | 状态转移方程验证 |
| 典型例子 | 硬币问题（特定面值） | 硬币问题（任意面值） |

## 贪心的基本步骤

```
1. 将问题分解为若干子问题
2. 确定贪心策略（如何做出当前最优选择）
3. 按策略逐步求解
4. 得到最终解
```
